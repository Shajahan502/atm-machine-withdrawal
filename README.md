# 🏧 ATM Machine – Cash Withdrawal System

## 📌 Project Overview
This project simulates an ATM machine that allows users to withdraw money securely using PIN authentication.

It demonstrates how basic banking operations work in a simple and interactive way using Python.

---

## 🎯 Features
- 🔐 PIN Authentication
- 💰 Balance Checking
- 💸 Cash Withdrawal
- ❌ Insufficient Balance Handling
- 🔁 Continuous Transactions until Exit

---

## 🧠 Concepts Used
- Conditional Statements (if-else)
- Loops (while)
- Functions
- User Input Handling

---

## 💻 Technologies Used
- Python 3

---

## 📂 Project Structure

# atm-machine-withdrawal
ATM Cash Withdrawal Prediction System using Python, where historical transaction data is analyzed using Linear Regression to predict future withdrawals, along with graphical visualization of results.
